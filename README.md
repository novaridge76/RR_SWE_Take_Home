# DayMark

Initial monorepo scaffold for a daily mood check-in app.

## Setup

```bash
npm install
npm run dev
```

- Client: http://127.0.0.1:5173/
- GraphQL API: http://localhost:4000/

## Layout

- `client/` — React + TypeScript (Vite)
- `server/` — Apollo GraphQL API (health check only for now)
